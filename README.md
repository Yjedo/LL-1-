# LL1-and-LR1
前后端分离的LL(1)语法分析器
前后端分离的LR(1)语法分析器

LLBack文件夹与LLFront文件夹分别为LL(1)的前后端项目，其中还包含了初始版本的LR(1)

LR(1)文件夹为初始LR(1)的优化版本，其优化了代码结构、规范了命名、添加了一定注释
同时集成了mysql、redis对文法相关信息进行持久化与缓存。还优化了前端的数据展示，添加了项目集族的展示
